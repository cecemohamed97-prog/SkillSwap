 SkillSwap

This project is a simple website that I made to help people exchange skills with each other.

 Project Idea
The idea came from the problem that many people want to learn new skills but courses are expensive. 
So I thought about making a platform where people can teach what they know and learn from others without paying.

Features
- Users can exchange skills
- Easy and simple to use
- No need to pay money
- Users can connect with each other

 Technologies Used
- HTML
- CSS
- Bootstrap
- JavaScript
